Projetos

RocketNotes

Esta é uma aplicação fullstack que permite salvar e gerenciar links úteis de forma rápida. Criei uma API com Node.js, Express e SQLite para o backend, enquanto o frontend foi desenvolvido em ReactJS com Styled Components.

RocketMovies

Esta uma aplicação é uma plataforma permite acompanhar filmes, séries e animes assistidos. Utilizei Vite, ReactJS, styled components, Node.js, Express e SQLite, com recursos como autenticação JWT, upload de imagens e API RESTful.

Gitfav

O projeto Git Fav é uma aplicação web que permite aos usuários criar uma lista personalizada de favoritos do GitHub. Utilizei a API pública do GitHub para buscar informações relevantes dos usuários, como imagem, nome, repositórios e seguidores.

Pomodoro Timer

O Pomodoro Timer é uma aplicação web de gerenciamento de tempo, desenvolvida com HTML, CSS e JavaScript. Permite aos usuários personalizar o tempo de foco, ajustar o volume de áudio e utilizar áudios pré-definidos para melhorar a produtividade.
